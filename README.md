# Skill Lookup

This program looks up a skill and prints its name, cooldown, and level.

## Requirements

- Python 3

## How to Run

- Clone or download this repository into same folder.
- Open skills_lookup.py and run. You will be prompted to type a skill name or quit.

## Example

Skill name(or quit): Fireball
Fireball has a cooldown of 300 and a level of 30
Skill name(or quit): Blink
Blink has a cooldown of 500 and a level of 35
